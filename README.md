# متجر العالمية (Al Alamia Shop)

تطبيق تجارة إلكترونية متكامل مبني باستخدام Flutter، يهدف لتوفير تجربة تسوق سلسة ومميزة.

## معلومات النظام
* **اسم التطبيق:** متجر العالمية
* **إصدار التطبيق:** 16.0
* **إصدار Flutter SDK المدعوم:** 3.38.5
* **إصدار تطبيق المناديب (Deliveryman):** 4.8
* **رابط الموقع الأساسي (Base URL):** [https://alalamia.shop](https://alalamia.shop)

## المميزات الرئيسية
* نظام تسوق متكامل (تصنيفات، علامات تجارية، منتجات).
* دعم كامل للغة العربية والإنجليزية ولغات أخرى.
* نظام محادثة مباشر مع البائعين والمناديب.
* تتبع الطلبات بشكل مباشر.
* دعم المحفظة الإلكترونية ونقاط الولاء.
* طرق دفع متعددة (رقمي، يدوي، محفظة).

## متطلبات التشغيل
للبدء في تشغيل المشروع، تأكد من تثبيت المتطلبات التالية:
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.38.5)
- Android Studio / VS Code
- إعداد ملف `google-services.json` (تم إعداده مسبقاً في هذا المشروع).
