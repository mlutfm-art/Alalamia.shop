## 📢 1. موديول الإعلانات الذكية (SmartAds Module)

تم دمج وتفعيل الإعلانات الذكية في الصفحة الرئيسية للتطبيق بشكل ديناميكي كامل.

### 💡 طريقة العمل (How it works):
1. **اللافتات المصورة (Banners):**
   التطبيق يقوم بطلب كافة الإعلانات النشطة من السيرفر دفعة واحدة عبر مسار موحد `/get-active/all` لمنع تعدد الطلبات. يتم توزيع وعرض الإعلانات محلياً في الشاشة الرئيسية حسب المعرفات التالية:
   * **لافتة علوية (`home_top`):** تظهر مباشرة تحت سلايدر البانرات الرئيسي.
   * **لافتة وسطى (`home_middle`):** تظهر مباشرة فوق قسم المنتجات المقترحة.
   
2. **الإعلانات المنبثقة (In-App Popups):**
   عند فتح التطبيق/الصفحة الرئيسية، يتم التحقق تلقائياً من وجود إعلانات منبثقة موجهة للمستخدم. وفي حال وجودها، تظهر نافذة حوارية منبثقة تلقائية (`InAppBannerDialog`) مع زر اتخاذ إجراء للذهاب للرابط.

3. **تتبع الإحصائيات (Impressions & Clicks Tracking):**
   * بمجرد ظهور الإعلان على شاشة المستخدم، يقوم التطبيق بإرسال طلب تتبع تلقائي للخلفية لتسجيل ظهور جديد (Impression).
   * عند نقر العميل على الإعلان، يتم تسجيل نقرة جديدة (Click) وفتح الرابط المرفق بالإعلان.

## 🤖 2. موديول البوت الذكي (BotCenter Module)

تم ربط شاشة المحادثة مع البوت الذكي وتفعيل جلب البيانات الحقيقية من قاعدة البيانات.

### 💡 طريقة العمل (How it works):
1. **الخدمات والردود السريعة (Quick Actions):**
   يتم جلب النوايا ومداخل المعرفة النشطة من قاعدة البيانات ديناميكياً وعرضها في واجهة الترحيب بالبوت كـ "قائمة خدمات سريعة"، لكي ينقر العميل على الخدمة التي يحتاجها مباشرة دون الحاجة للكتابة.
2. **استقرار الـ API:**
   تم تغليف معالجة طلب البوت في الخلفية لتفادي أخطاء الـ 500 واستخدام معرّفات الجلسات المتغيرة لتتبع المحادثة.

## 🚀 4. أوامر تشغيل وبناء التطبيق (Run & Build Commands)

شغّل الأوامر التالية من خلال الـ Terminal المدمج في أندرويد ستديو لتطبيق التحديثات:

### لتنظيف الكاش وتحميل الاعتماديات الجديدة:
```bash
flutter clean
flutter pub get
```

### لتشغيل التطبيق في وضع التطوير (Debug Mode):
```bash
flutter run
```

### لبناء نسخة الأندرويد النهائية (Release APK):
```bash
flutter build apk --release
```
